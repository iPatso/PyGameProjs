English
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
The game is started by running the module "source/Game.py."

In the folder "source/diagrams" has two simplified diagrams of the Game 'SimpleClassDiagram' which is a simplified class diagram and 'SimpleGraphFlow' which shows the basic flow of the methods of the game.

This game was made and tested with "python-2.7.2" and "pygame-1.9.2"
